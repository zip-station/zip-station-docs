# Canned Responses

Canned responses are pre-written reply templates that save your team time on common questions.

## Creating a Template

1. Go to **Canned Responses** from the sidebar
2. Click **New Response**
3. Give it a **title** (e.g., "Password Reset Instructions")
4. Write the response body using the **rich text editor** — you can format text, add links, and include lists
5. Save the template

## Using Canned Responses

When replying to a ticket:

1. Click the **canned response** button in the reply toolbar
2. Search or browse your templates
3. Select one — the content is inserted into your reply
4. Edit the text as needed before sending

## Tips

- Keep templates general enough to reuse, but specific enough to be helpful
- Use canned responses for greetings, common troubleshooting steps, and closing messages
- Your whole team shares the same canned response library
