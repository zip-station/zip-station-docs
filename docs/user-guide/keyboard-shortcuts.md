# Keyboard Shortcuts

Navigate Zip Station quickly using keyboard shortcuts. Press the keys in sequence (not simultaneously).

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `g` then `d` | Go to Dashboard |
| `g` then `t` | Go to Tickets |
| `g` then `i` | Go to Intake |
| `g` then `c` | Go to Customers |
| `g` then `p` | Go to Projects |
| `g` then `s` | Go to Settings |
| `g` then `a` | Go to Audit Log |

## How It Works

Press `g` first, then press the second key within a short window. For example, to navigate to Tickets, press `g`, release it, then press `t`.

These shortcuts work from anywhere in the dashboard, as long as you're not focused on a text input field.
